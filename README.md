# nlp-playground

## Common Regex Patterns
| pattern | matches         | example     |
| ------- | --------------  | -------     |
| \w+     | word            | 'Magic'     |
| \d      | digit           | 9           |
| \s      | space           | ''          |
| .*      | wildcard        | 'username74'|
| + or *  | greedy match    | 'aaaaa'     |
| \S      | *not* space     | 'no_spaces' |
| [a-z]   | lowercase group | 'abcdefg'   |
